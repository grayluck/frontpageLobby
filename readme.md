## Installation

1. Install NodeJS and MySQL Server beforehand.
2. At the root directory, use the command `npm install`
3. Change the username and password to the database in `server/env.js`
4. Create database in MySQL. The name of the database should be corresponding to the name set in `server/env.js`
5. `npm start`
6. Access to `localhost:3000`. You should be guided to the login page.
